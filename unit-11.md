## 1. Main topics

This material covers household appliances, the purpose of everyday objects, encouraging purchases, product description, requests, aspect after negation, impersonal forms, deverbal nouns, and comparison of life in the past and the present.

## 3. Expressing requests

Polite requests can be formed in several common ways. Natural patterns include asking directly with **czy możesz**, softening the request with **czy mógłbyś / mogłabyś**, asking for information, or asking to borrow something.

### 3.1 Main request patterns

- **Czy możesz + infinitive?** 
- **Czy mógłbyś / mogłabyś + infinitive?** 
- **Pożyczyłabyś mi... ?** 
- **Czy możesz mi powiedzieć...?** 

### 3.2 Examples

- **Czy możesz zadzwonić do serwisu?** 
- **Czy mógłbyś włączyć telewizor?** 
- **Czy możesz mi powiedzieć, jak to działa?** 
- **Pożyczyłabyś mi suszarkę?** 

## 4. Aspect after negation

A key rule concerns negated infinitives after **móc**. After **móc + nie**, Polish normally uses the imperfective infinitive.

### 4.1 Rule

**Móc + nie + infinitive** usually requires the **imperfective** form, not the perfective form.

### 4.2 Basic contrast

| Positive request | Negative request |
|---|---|
| Czy możesz wyłączyć komputer?  | Czy możesz nie wyłączać komputera?  |
| Mogę zrobić kawę.  | Możesz nie robić tyle hałasu?  |

### 4.3 Common transformations

| Positive form | Negative form |
|---|---|
| włączyć | nie włączać  |
| wyłączyć | nie wyłączać  |
| dać | nie dawać  |
| zadzwonić | nie dzwonić  |
| naprawić | nie naprawiać  |
| zrobić | nie robić  |

### 4.4 Model sentences

- **Czy możesz nie wyłączać komputera? Wychodzimy już!** 
- **Czy mógłbyś nie dawać dziecku pilota?** 
- **Czy możesz nie włączać teraz odkurzacza? Oglądam telewizję.** 
- **Czy możesz nie dzwonić jeszcze do serwisu?** 
- **Czy mógłbyś nie naprawiać tego DVD? Jest na gwarancji.** 
- **Czy mogłabyś nie wyłączać radia?** 

## 5. Impersonal forms

Impersonal forms describe actions in a general way, without naming one specific person. They are built with the third person singular form and **się**, for example **kupować → kupuje się**.

### 5.1 Rule

**Verb in the 3rd person singular + się** expresses a general activity: what people do, what is commonly done, or what happens in general. A standard pattern is **kupować (on kupuje) → kupuje się**.

### 5.2 Meaning

This structure commonly means:
- people do...
- one does...
- it is done...

### 5.3 Present tense examples

- **W Polsce mieszka się w blokach.** 
- **W dużych miastach robi się zakupy w supermarkecie.** 
- **W polskich szkołach uczy się języków obcych.** 
- **Teraz płaci się kartą.** 

### 5.4 Past tense examples

- **Dawniej kupowało się jedzenie na kartki.** 
- **Na telefon czekało się 10 lat.** 
- **Żeby kupić lodówkę, trzeba było stać w kolejce.** 

### 5.5 Common impersonal expressions

| Present | Past |
|---|---|
| jest / są  | było / były  |
| nie ma  | nie było  |
| trzeba  | trzeba było  |
| można  | można było  |
| wolno  | wolno było  |
| trudno  | trudno było  |
| łatwo  | łatwo było  |

## 6. Deverbal nouns

Deverbal nouns are nouns formed from verbs. They are common in Polish and often appear after **do** or as names of activities.

### 6.1 Formation

| Verb | Deverbal noun |
|---|---|
| czytać | czytanie  |
| rozumieć | rozumienie  |
| uczyć | uczenie  |
| robić | robienie  |
| być | bycie  |
| myć | mycie  |
| zdjąć | zdjęcie  |
| nosić | noszenie  |
| chodzić | chodzenie  |

### 6.2 With objects

- **czytać książkę → czytanie książki** 
- **książka do czytania** 
- **pisać list → pisanie listu** follows the same pattern used in the exercises.
- **pić kawę → picie kawy** follows the same pattern used in the exercises.
- **nosić okulary → noszenie okularów** follows the same pattern used in the exercises.

### 6.3 With purpose expressions

A common pattern is **służyć do + deverbal noun in the genitive**.

## 10. Grammar summary

### 10.1 Expressing purpose

Use **żeby + infinitive** to say why an object is bought or used, for example **Kupujemy czajnik, żeby gotować wodę**.

### 10.2 Polite requests

Use **Czy możesz... ?**, **Czy mógłbyś / mogłabyś... ?**, and similar patterns to ask for help, permission, or information politely.

### 10.3 Negation after móc

Use the **imperfective infinitive** after **móc + nie**, for example **Czy możesz nie wyłączać komputera?**.

### 10.4 Impersonal forms

Use **3rd person singular + się** to describe actions done in general, for example **W Polsce płaci się kartą** and **Dawniej kupowało się jedzenie na kartki**.

### 10.5 Deverbal nouns

Form nouns from verbs to name activities, for example **czytać → czytanie**, **myć → mycie**, and **prasować → prasowanie**. These nouns often appear after **do**, as in **do prasowania**.
